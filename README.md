# Java Generics

## Description
Practice working with **generic** and **non-generic methods** in Java.  
These exercises help you understand:

- How to store and manipulate objects of the same type.
- How to create generic methods that accept any type of parameter.
- How type safety and flexibility can be achieved with generics.

---

## Exercise 1: Non-Generic Class

**Goal:**  
Create a class `NoGenericMethods` that:

- Stores **three arguments of the same type**.
- Provides methods to **store** and **retrieve** these objects.
- Includes a **constructor** to initialize all three objects.
- Allows arguments to be passed in **any order** when calling the constructor.

---

## Exercise 2: Generic Method

**Goal:**  
Create a class `Persona` with attributes:

- `nombre`  
- `apellido`  
- `edad`  

Then, create a class `GenericMethods` with a **generic method** that:

- Accepts **three arguments of any type**.
- Prints the arguments to the console.  

**Example Usage in `main()`:**

- A `Persona` object  
- A `String`  
- A primitive type
